Weather and Clocks App

What is it?

It is a simple local weather and world clocks app i made with HTML, CSS and Javascript that utilizes APIs and Geolocalization to get the exact time at different timezones and get your local weather.


I used the fetch API and promises to get my data from the two online APIs that i used to get the data about the local weather using the browser's Geolocation to get your latitude and longitude that was needed for the API query


Once again I used the flexbox and Google Fonts when styling this app and it's amazing how the flexbox makes it quick and easy to position everything where i wanted it to be.
