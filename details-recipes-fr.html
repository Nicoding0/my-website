Recherche de recettes

C'est quoi?

J'ai creer cette application avec ReactJS pour solidifier ce que j'apprenais, j'ai utiliser un API de recherche de recettes qui retourne des recettes contenant l'ingrédient que tu écris et soumets depuis la barre de recherche.


J'ai utiliser les "hooks" (useState, useEffect...) qui ont été introduit avec la version 16.8 de ReactJS qui remplacent les classes ainsi que le React Router pour remplacer les lien "a" qui viennent de HTML.


Ici j'ai utiliser le style en ligne et les modules CSS pour styler mes composant React et je doit bien avouer que c'est bien plus intuitif et rapide de styler comme ceci car avec le style en ligne tout est dans le même fichier/composant.
