Snake

What is it?

This is a classic snake game I made using HTML, CSS and Javascript, it uses the HTML5 Canvas to draw the game on.


I used an object oriented approach to keep every functions separate from the main Javascript file, like drawing the snake, updating it and drawing the food on the canvas.


I used the flexbox a lot when styling the game, i found it really easy to position the different elements where i wanted them using it and it has the advantage to be responsive but my game is not really suited for mobile.
