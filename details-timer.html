Cumulative Timer

What is it?

I wanted to keep track of how much time I spend while programming so I made this little timer in Javascript, HTML and CSS


There's not much going on it counts up from seconds to minutes, hours and days but I didn't want the timer to get reset everytime I close the page so I used local storage to store the time I spend programming so that I get the total time. It could obviously be used for other purposes but I originally made it for keeping count of the hours I spend.


Not much is to be said about the styling, it is basic, I used the flexbox to center my elements with justify-content and align-items, other than that it's pretty standard CSS.
