Snake

C'est quoi?

C'est un jeu Snake classique que j'ai créer avec HTML, CSS et Javascript il utilise l'élément canvas introduit avec HTML5 pour dessiner le jeu.


J'ai utilisé une approche orienté objet pour garder toutes les fonctions séparé du fichier Javascript principal, comme dessiner le serpent, le mettre a jour et placer la nourriture sur le canvas


J'ai beaucoup utiliser le module de boite fléxible ou "Flexbox" en stylant le jeu et j'ai trouver vraiment simple a placé tout les éléments ou je les voulais, ca a aussi l'avantage d'être résponsif mais mon jeu n'est pas vraiment adapté pour les mobiles.
