Recipe search

What is it?

I created this application with ReactJS to solidify what I was learning, I used a recipe search API which returns recipes containing the ingredient you wrote and submitted using the search bar.


I used React Hooks (useState, useEffect...) which were introduced with version 16.8 of ReactJS that replace classes as well as the React Router to replace the "a" tag links that come from HTML.


Here I used inline styling and CSS modules to style my components and I must admit that it's much more intuitive and faster to style like this because with inline style everything is in the same file/component.
