Lyrics Search

What is it?

I decided to create this application with ReactJS to be more comfortable and continue my learning of this Javascript library.


I wanted to learn and use the Context API which is integrated into React since version 16.3 which makes it much easier to maintain our components and our properties inheritance.


For this application I wanted to use and learn Bootstrap, the CSS kit created by the developers of Twitter which allows to create aesthetic and responsive websites more easily and more quickly.
